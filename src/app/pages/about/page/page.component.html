ABOUT SHIELD

Surveillance Health Information Exchange Linkage Data (SHIELD) is a centralised data repository developed by the Ministry of Health (MOH) for surveillance data and resources.

Key Features:

  • Data Integration: SHIELD seamlessly integrates diverse health data sources, including Kenya Health Information System (KHIS-2), electronic health records (EMRs), laboratory Information Systems (LMIS), etc., to create a comprehensive surveillance database. Data from different data sources can be ingested into SHIELD using Application Programming Interfaces (APIs) or uploaded manually through the Upload page.
  • Analytics: The surveillance data in SHIELD is analyzed and visualized through dashboards to facilitate data-driven decision-making. The dashboards available on SHIELD include the Routine Surveillance dashboard, Event-Based Surveillance dashboard, Outbreak Response Dashboards, and Program dashboard.
  • Real-time Monitoring: SHIELD platform offers real-time monitoring capabilities, allowing health professionals to track health indicators, monitor disease prevalence, and respond promptly to emerging health threats.
  • Secure Exchange: SHIELD prioritizes data security and privacy, ensuring compliance with the highest standards of confidentiality and data protection. Our secure exchange protocols facilitate safe and efficient data sharing among authorized stakeholders.
  • Resources: The SHIELD platform provides an array of resources tailored to meet the diverse needs of health surveillance professionals. This includes guidelines, standard operating procedures, frameworks, etc. The resources can be found on the Resources Page.
